# LNEH Premium Commercial Website

GitHub/Vercel-ready static website for LNEH Natural Herbal Skincare.

## Included
- Homepage matching premium botanical skincare style
- Demo shop page
- Product detail pages
- About, Approach, Ingredients, Research, Contact, FAQ
- Blog sample page
- Logo asset from uploaded LNEH logo
- 20+ custom SVG visual assets and icons
- Responsive CSS and simple JS interactions
- `vercel.json` for clean deployment

## Run locally
Open `index.html` directly, or run:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Upload to GitHub
1. Create a new GitHub repository.
2. Upload all extracted files from this ZIP.
3. Commit to the `main` branch.

## Deploy on Vercel
1. Go to Vercel.
2. Add New Project.
3. Import the GitHub repository.
4. Keep framework preset as Other/Static.
5. Click Deploy.

## Important
This is a premium demo site. Review all product descriptions, prices, legal pages, policies, and cosmetic claims before public commercial launch.
